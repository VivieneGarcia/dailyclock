## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `1` | Clock |
| `2` | Timer |
| `3` | Stopwatch |
| `Space` | Play / Pause |
| `R` | Reset |
| `L` | Lap |
| `F` | Fullscreen |
| `D` | Toggle theme |
| `W` | Prevent sleep |
| `S` | Settings |
| `Esc` | Close settings |
